Pattern Library: CSS Examples

The following patterns are available as class names in the patterns.css file. Here's what a classname might look like:

          
            .circles-1 {
              background-image: url("data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' width='10' height='10'><path fill='#fff' d='M0 0h10v10h-10z'/><circle cx='1' cy='1' r='1'/></svg>");
              background-repeat: repeat;
            }
